# Design Rationale (เหตุผลในการออกแบบ)

## 1. Layout (การจัดวาง)
เลือกใช้ **Grid Layout** ขนาด 4 คอลัมน์ ซึ่งเป็นมาตรฐานสากลของเครื่องคิดเลข ทำให้ผู้ใช้เรียนรู้การใช้งานได้ทันที (Intuitive Design) โดยวางหน้าจอแสดงผลไว้ด้านบนสุดเพื่อให้เห็นชัดเจน

## 2. Color Scheme (ชุดสี)
* **พื้นหลัง (Background):** สีเทาเข้ม (#222) เพื่อถนอมสายตา และลดแสงสะท้อนขณะใช้งาน
* **ปุ่มตัวเลข:** สีเทาอ่อน เพื่อให้ตัดกับพื้นหลังและอ่านง่าย
* **ปุ่มเครื่องหมาย (+ - * /):** สีส้ม (#ff9500) เพื่อสร้างจุดสนใจ (Call to Action) แยกฟังก์ชันคำนวณออกจากตัวเลข
* **ปุ่มล้างค่า (C/DEL):** สีแดงหรือสีที่แตกต่าง เพื่อเตือนผู้ใช้ว่าเป็นปุ่มลบข้อมูล

## 3. Typography (ตัวอักษร)
ใช้ Font ขนาดใหญ่ (เช่น Sans-Serif) เพื่อให้อ่านตัวเลขได้ชัดเจน ลดความผิดพลาดในการมองเห็น
