# คู่มือการใช้งาน (User Manual)

## วิธีการใช้งาน
1. **การเปิดโปรแกรม:** เข้าไปที่โฟลเดอร์ `04_Implementation/src` แล้วคลิกไฟล์ `index.html`
2. **การคำนวณ:** - คลิกปุ่มตัวเลขเพื่อใส่ค่า
   - เลือกเครื่องหมาย (+ - * /)
   - กดปุ่ม = เพื่อดูผลลัพธ์
3. **การลบค่า:**
   - ปุ่ม **DEL**: ลบตัวเลขตัวล่าสุด
   - ปุ่ม **C**: ล้างค่าทั้งหมดเพื่อเริ่มใหม่
