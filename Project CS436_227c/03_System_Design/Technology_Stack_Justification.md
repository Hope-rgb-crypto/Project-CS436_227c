# Technology Stack Justification

## 1. HTML5 (HyperText Markup Language)
* **เหตุผล:** เป็นมาตรฐานหลักในการสร้างโครงสร้างเว็บ ใช้งานง่าย และรองรับทุกเบราว์เซอร์โดยไม่ต้องติดตั้งโปรแกรมเสริม

## 2. CSS3 (Cascading Style Sheets)
* **เหตุผล:** ใช้สำหรับตกแต่งหน้าตา Application ให้สวยงาม สามารถทำ Responsive Design (Grid/Flexbox) ให้รองรับหน้าจอมือถือได้ง่าย

## 3. JavaScript (Vanilla JS)
* **เหตุผล:** เหมาะสำหรับ Logic การคำนวณที่ไม่ซับซ้อน ประมวลผลได้ทันทีที่ฝั่ง Client (Client-side Scripting) ทำให้ระบบทำงานเร็วมากโดยไม่ต้องมี Server หรือ Database ลดความซับซ้อนของระบบตามโจทย์ Simple Calculator
